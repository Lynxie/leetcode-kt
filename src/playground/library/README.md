# Мини-библиотека

Нужно написать небольшую программу для управления библиотекой.

## Что должно быть в программе

- Книга: название, автор, ISBN и жанр.
- Читатель: имя и список взятых книг.
- Библиотека: список книг и читателей.
- Книгу можно выдать читателю и вернуть обратно.
- Нельзя выдать книгу, если она уже у другого читателя.
- Один читатель не может взять больше трёх книг.
- Можно найти книги по автору или жанру.
- Можно вывести список всех доступных книг.

## Что стоит применить

- `class` и `data class`;
- `enum class` для жанров;
- инкапсуляцию и приватные поля;
- nullable-типы;
- `when`;
- операции с коллекциями;
- `sealed class` для результата выдачи или возврата книги;
- хотя бы одну extension-функцию.

## Первый этап

Сначала создай модели книги и читателя, а также класс библиотеки. Реализуй добавление книг и регистрацию читателей. После этого переходи к выдаче и возврату книг.

Готовое решение в этой папке создавать не нужно — реализацию напиши самостоятельно.
